# VOICE ASSISTANT

#### Python Library
pyttsx3

SpeechRecognition

pyaudio


## INSTALL ON TERMINAL

[ INSTALLING PYTTSX3 ]

```bash
pip install pyttsx3
```
---------

[ INSTALLING SPEECH RECOGNITION ]

```bash
pip install SpeechRecognition
```

---------

[ INSTALLING PYAUDIO ]

```bash
pip install Pyaudio
```

---------

git clone https://github.com/JOVANEAH/Python-Voice-Assistant
